# origin-car-medical 技术文档

## 项目概述

基于 ROS 2 Humble 的 Origincar Ackermann 底盘自主导航堆栈，适配竞赛固定场地路线。

## 系统架构

```
传感器层               定位层                  规划层                 控制层
┌──────────┐    ┌─────────────────┐    ┌──────────────────┐    ┌──────────────┐
│ LSLIDAR  │───▶│ slam_toolbox    │    │ Smac Hybrid-A*   │    │ MPPI 控制器  │
│ N10      │    │ (定位模式)      │    │ (全局规划)       │    │ (局部轨迹)   │
├──────────┤    ├─────────────────┤    ├──────────────────┤    ├──────────────┤
│ BMI088   │───▶│ robot_localization│   │ AckermannRoute   │    │ CruisePath   │
│ IMU      │    │ EKF (15状态)    │    │ (路径生成)       │    │ ManeuverPath │
├──────────┤    ├─────────────────┤    └──────────────────┘    ├──────────────┤
│ 底盘里程计│───▶│ ekf_sensor_     │                            │ Ackermann    │
│          │    │ adapter         │                            │ Converter    │
└──────────┘    └─────────────────┘                            └──────────────┘
```

## 快速启动

```bash
# 完整竞赛
bash scripts/start_stack.sh

# 简化竞赛
bash scripts/start_simple_ekf_cruise.sh

# 仅定位测试
bash scripts/start_lidar_localization_test.sh

# 配置验证
python src/tools/validate_config.py

# 运行测试
python -m pytest tests/
```
