队伍：从零开始的调车生活，队长：任崇涛，队员：祝鑫铭，刘富钰，刘晓琪
本次比赛是需要到达特定位置执行特定任务，同时需要注意规避路上的其他障碍物，所以我们的思路是首先我们进行定位来执行巡航，在巡航的同时我们要执行避障，我们初期的方案是希望使用scan进行全局定位的，但是由于周围屏障相同，很难断定自己此时的位置，所以我们需要进行重定位，让小车自己在预先构建的地图里定位自己的位置，从而反馈自己的位置，然后这里我们使用了slam技术，并且针对于雷达漂移的情况进行了特定处理，同时，我们使用nav2的MPPI和HybridA*进行局部和全局规划，通过nav2，并同时在代码里部署相关逻辑，我们实现了巡航和避障功能，
